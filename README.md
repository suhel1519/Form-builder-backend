# Form Builder Backend

REST API for the dynamic form builder system.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm start
```

## API Endpoints

- `GET /api/form-schema` - Get form schema
- `POST /api/submissions` - Submit form data
- `GET /api/submissions` - Get paginated submissions
